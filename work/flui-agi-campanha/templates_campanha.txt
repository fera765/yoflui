# Templates de Campanha - FLUI AGI

## Diretrizes Visuais para Materiais de Marketing

### 1. Template de Email Marketing

**Cabeçalho:**
- Logo FLUI AGI (posição superior esquerda)
- Cores: Azul escuro (#1a365d) e branco
- Fonte: Inter, tamanho 16px para corpo

**Estrutura do Email:**
```
[Logo FLUI AGI]
[Data de envio]
[Preheader: Visão rápida do conteúdo]

[Cabeçalho principal com título da campanha]
[Imagem de destaque ou banner]
[Texto principal com call-to-action]
[Botão de ação destacado]
[Footer com informações de contato]
```

**Cores Padrão:**
- Primária: #3182ce (azul)
- Secundária: #2b6cb0 (azul escuro)
- Destaque: #38a169 (verde)
- Texto: #2d3748 (cinza escuro)
- Fundo: #ffffff (branco)

### 2. Template de Post para Redes Sociais

**Dimensões Padrão:**
- Instagram: 1080x1080px
- LinkedIn: 1200x627px
- Twitter: 1200x675px

**Elementos Visuais:**
- Logo FLUI AGI (canto superior direito, 10% do tamanho da imagem)
- Tipografia: Montserrat para títulos, Open Sans para corpo
- Espaçamento: 24px entre seções
- Bordas arredondadas: 8px

**Layout Sugerido:**
```
[Imagem de fundo com gradiente azul]
[Ícone representativo da funcionalidade]
[Título em destaque]
[Subtítulo explicativo]
[Chamada para ação]
[Hashtags relevantes]
```

### 3. Template de Apresentação (Pitch Deck)

**Estrutura Padrão:**
1. Capa (Logo, título, data)
2. Sumário
3. Problema
4. Solução
5. Diferenciais
6. Mercado
7. Modelo de negócios
8. Equipe
9. Resultados
10. Próximos passos

**Paleta de Cores:**
- Fundo: #ffffff ou #f7fafc
- Texto principal: #1a202c
- Texto secundário: #4a5568
- Destaques: #3182ce
- Acentos: #ecc94b

**Fontes:**
- Títulos: Inter Bold, 32-48px
- Subtítulos: Inter SemiBold, 24px
- Corpo: Inter Regular, 18-20px
- Legendas: Inter Light, 14px

### 4. Template de Landing Page

**Seções Obrigatórias:**
- Header com navegação
- Hero section com CTA
- Features principais
- Testemunhos
- Pricing (se aplicável)
- FAQ
- Footer

**Elementos de Design:**
- Espaçamento generoso (48px entre seções)
- Cards com sombras leves (box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1))
- Botões com bordas arredondadas (8px)
- Animações sutis no hover

### 5. Template de Material de Apoio (PDFs, Whitepapers)

**Formato Padrão:**
- Tamanho: A4 (210x297mm)
- Margens: 25mm em todos os lados
- Colunas: 1 ou 2 (dependendo do conteúdo)

**Elementos Gráficos:**
- Cabeçalho com logo e título
- Índice com links
- Seções com divisórias
- Gráficos e tabelas padronizados
- Rodapé com informações de contato

### 6. Diretrizes de Tipografia

**Fontes Principais:**
- Títulos: Inter Bold
- Subtítulos: Inter SemiBold
- Corpo: Inter Regular
- Códigos: JetBrains Mono

**Hierarquia de Texto:**
- H1: 36px, bold
- H2: 28px, semi-bold
- H3: 22px, semi-bold
- Parágrafos: 16px, regular
- Legendas: 14px, light

### 7. Diretrizes de Cores

**Paleta Primária:**
- Azul Escuro: #1a365d
- Azul Médio: #3182ce
- Azul Claro: #63b3ed
- Verde: #38a169
- Amarelo: #ecc94b
- Laranja: #dd6b20
- Vermelho: #e53e3e

**Cores de Fundo:**
- Branco: #ffffff
- Cinza Claro: #f7fafc
- Cinza Médio: #e2e8f0
- Cinza Escuro: #4a5568

### 8. Elementos de Interface

**Botões:**
- Primário: Azul (#3182ce), texto branco
- Secundário: Branco (#ffffff), borda azul
- Terceário: Transparente, texto azul

**Cards:**
- Fundo branco
- Borda: 1px solid #e2e8f0
- Sombra: 0 1px 3px 0 rgba(0,0,0,0.1)
- Bordas arredondadas: 8px

**Formulários:**
- Campos com borda: 1px solid #cbd5e0
- Foco: anel azul (#3182ce)
- Erros: borda vermelha (#e53e3e)

### 9. Elementos de Animação

**Transições Padrão:**
- ease-in-out, 0.3s
- Transformações suaves
- Opacidade gradual

**Micro-interações:**
- Hover em botões: escala 1.05
- Focus em inputs: borda azul
- Loading: spinner azul

### 10. Diretrizes de Acessibilidade

- Contraste adequado (WCAG 2.1 AA)
- Tamanhos de fonte legíveis
- Alternativas de texto para imagens
- Navegação por teclado
- Suporte a leitores de tela
